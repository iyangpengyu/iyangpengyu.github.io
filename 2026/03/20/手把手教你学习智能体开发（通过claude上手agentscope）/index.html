手把手教你学习智能体开发（通过claude上手agentscope）

准备步骤（下载、配置跳过登陆、准备国内模型token）

下载claude code

mac: curl -fsSL https://claude.ai/install.sh | bash
window: curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd

安装后验证

claude –version
如果显示版本号则安装成功

登陆跳过

安装完毕后，会一直让用户登陆，工具在启动时首先检查用户是否已经完成“入门流程”（onboarding），而不是直接检查可用的 API 凭证。这意味着，即使你的 API 设置完全正确，只要系统认为你还没有完成初始设置，就会强制你进入登录流程。
此时需要执行如下操作：
检查配置文件：ls -la ~/.claude.json
不存在：echo’{“hasCompletedOnboarding”: true}’ > ~/.claude.json
存在则直接修改：

选用国内模型token

例如kimi,博主用的deepseek，感觉还是蛮费钱的，可以冲个10块玩下。
Configure the following content in ~/.claude/settings.json to skip logging in and use the kimi api

{
“env”: {
“ANTHROPIC_API_KEY”: “xxx”,
“ANTHROPIC_BASE_URL”: “https://api.moonshot.cn/anthropic/“,
“CLAUDE_CODE_MAX_OUTPUT_TOKENS”: 32000,
“CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC”: 1
}
}

基于项目进行问答学习

直接ask，之前看过ai编码很强大，可以不了解代码直接由ai进行代码issue修复和feature完善。所以上来就想直接提PR，发现看了issue并不能看懂，所以就先从了解项目入手。
